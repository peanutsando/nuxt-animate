# Nuxtjs with animate.css

nuxtjs animation.css example

## Used

- npm: node-sass, sass-loader
- cdn: animate.css

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```

## Issue

_Speed_ class(delay, fast, slow..) did not work well
so CSS attribute was created and applied.

```
.slow {
  animation-duration: 5s;
}

.normal {
  animation-duration: 2s;
}

.fast {
  animation-duration: 0.8s;
}
```
